# LeedsUnitedTwitterBot
A Twitter bot for all things Leeds United.

This is a unfinished project. But is setup with the intentions of interacting with the fantasy premier league API and Twitter. It will interact with rhe FPL API and pull information about Leeds United players and then tweet the highest earning player for Leeds for the gameweek. Currently this must kicked of manually. It is all done in C#.

RoadMap

1:Intentions are to automate the project to send out tweets on a scheduled basis.
2:Allow users to tweet the twitter bot account questions. Which it will respond with.
